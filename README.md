#~~~~~~~~~~~~  Get Started  ~~~~~~~~~~~
#/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\
```
npm install
```

In one terminal tab, type:
```
gulp
```
This runs the browserify/reactify/coffee transforms for the client.

In another terminal tab, type:
```
npm start
```
This runs the Express server on port 3000

Then fire up

```
http://localhost:3000/#
```


Open to pull requests!
