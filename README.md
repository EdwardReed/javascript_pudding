#~~~~~~~~~~~~  Get Started  ~~~~~~~~~~~
#/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\/\

##Install NPM Packages
```
npm install
```

###Run browserify/reactify/coffee transforms for the client
```
gulp
```

### Run Express server on port 5000
In another terminal tab, type:
```
npm start
```
